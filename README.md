# News-RO-Offense

a novel Romanian language dataset for offensive message detection with manually 
annotated comment from a local Romanian news website (stiri de cluj) into five classes:

* non-offensive
* targeted insults
* racist
* homophobic
* sexist

Resulting in 4052 annotated messages
